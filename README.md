# Tripleten web_project_api_full

This full-stack application is a photo-sharing platform where users can upload images, view them in detail, and manage their own content. Users can also like photos posted by others.

The backend is built with Node.js, Express, and MongoDB, following a modular architecture and exposing a RESTful API. The frontend provides a responsive and interactive user experience.

Users can:

* Upload images (places or people)
* View images in an enlarged format
* Delete only their own uploaded images
* Like photos from other users

The backend is deployed at: **https://web-project-api-full-yhuw.onrender.com**

The frontend is deployed at:**https://web-project-api-full-zw3g.vercel.app**
